# Martial Arts: Decision Making Project  

## *Author: Hieu Nguyen*

## **HTML5 | CSS3 | Javascript | jQuery | Node | React | Redux | Third-Party API (YELP)**  

#### This is my decision making application on choosing the right martial arts. In this application once you answered the five questions, it will determine which discipline art to study and practice that can fit your needs. Make sure you turn on the location so the app can geolocate in your area.
